## 🎯 COMO USAR O AI STUDIO

### Setup Rápido (5 minutos)

1. **Acesse:** https://aistudio.google.com/

2. **Cole System Instructions:**
   - Abra: `RENTABILIZA-AI-STUDIO.md`
   - Copie **TODO** o conteúdo da seção "SYSTEM INSTRUCTIONS"
   - Cole no Google AI Studio (⚙️ Settings)

3. **Use Prompts:**
   ```
   Crie um Meta Ad completo para Rentabiliza:

   Objetivo: Captação de leads
   Público: Investidores 35-55 anos, patrimônio R$ 100k+
   Ângulo: Medo de investir sem análise
   CTA: Análise gratuita

   Forneça 3 variações + template visual completo.
   ```

### O Que o AI Studio Gera:

✅ **Copy profissional** (Headlines, Body, CTAs)
✅ **Especificações visuais detalhadas** (CSS, medidas)
✅ **Código HTML/CSS pronto** (emails, landing pages)
✅ **Prompts para IA de imagem** (DALL-E, Midjourney)
✅ **Brief para designers** (Figma, Canva)
