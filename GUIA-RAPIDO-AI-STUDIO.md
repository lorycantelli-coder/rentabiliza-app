## 🚨 ERROS COMUNS (EVITE!)

### ❌ Prompt Vago
```
"Crie um ad para Rentabiliza"
```
**Problema:** Muito genérico, a IA não sabe o contexto.

### ✅ Prompt Específico
```
"Crie Meta Ad para captar leads de investidores 40+ anos,
patrimônio R$ 200k+, que já investem mas sem análise técnica.
Ângulo: medo de estar perdendo ROI. CTA: análise gratuita."
```

---

### ❌ Pedir Muitos Formatos de Uma Vez
```
"Crie Meta Ad + Landing Page + Email + Google Ad"
```
**Problema:** Output muito longo e genérico.

### ✅ Um Formato por Vez
```
"Crie apenas Meta Ad. Depois peço Landing Page."
```
