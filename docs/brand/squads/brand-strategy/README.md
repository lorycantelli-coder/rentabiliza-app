## 🔄 WORKFLOWS

### **1. Create Brand Documentation**
Criar documentação completa de marca em YAML estruturado.

**Phases:**
1. **Research** → Analisar materiais existentes
2. **Strategy** → Definir identidade usando frameworks combinados
3. **Design** → Criar sistemas de identidade visual e verbal
4. **Documentation** → Gerar arquivos YAML estruturados
5. **Validation** → Quality check contra todos os frameworks

### **2. Validate Brand Identity**
Validar marca existente contra padrões mundiais.

**Phases:**
1. **Audit** → Avaliar estado atual
2. **Gap Analysis** → Identificar fraquezas usando todos os frameworks
3. **Recommendations** → Sugerir melhorias
4. **Implementation** → Guiar correções
