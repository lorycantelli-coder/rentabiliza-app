## 📝 REGRAS DE ESCRITA

### ✅ **SEMPRE Use:**

**1. Transparência Radical**
- Explique TODA decisão da IA
- "Por quê" antes de "Como"
- Dados concretos, não promessas

**2. Dados Primeiro, Promessas Nunca**
```
✅ "34% acima do IBOVESPA (média 36 meses)"
❌ "Retornos extraordinários garantidos"
```

**3. Educação Contínua**
- Ensine, não apenas execute
- Glossário inline para termos técnicos
- Analogias acessíveis quando necessário

**4. Vocabulário de Confiança**
- **Use:** Auditado, comprovado, transparente, explicável, validado
- **Evite:** Mágico, revolucionário, garantido, fácil, hack

---

### ❌ **NUNCA Use:**

**1. Promessas Irreais**
```
❌ "Fique rico rápido"
❌ "Retorno garantido de 50%"
❌ "Sem risco algum"
```

**2. Jargão Desnecessário**
```
❌ "Alpha generation através de smart beta strategies"
✅ "Retornos acima do mercado através de seleção inteligente de ativos"
```

**3. Urgência Artificial**
```
❌ "Últimas vagas! Corra!"
✅ "Novas matrículas abertas. 150 vagas disponíveis."
```

**4. Superlativos Vazios**
```
❌ "A melhor plataforma do universo"
✅ "34% acima do mercado. Auditado Ernst & Young."
```
